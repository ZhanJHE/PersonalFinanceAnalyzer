# 个人收支趋势分析器 - 项目起步文档 (VibeCoding 版)

## 🎯 项目目标

开发一个 **WPF 桌面应用**，实现个人收支的记录、分类、趋势分析与可视化，并通过**云端模型 API** 提供智能分析建议（如异常支出检测、消费习惯总结）。

**核心要求**：
- 连接 SQLite 数据库
- 使用 ScottPlot 展示图表（折线图、饼图）
- 调用云端 API（DeepSeek / Vercel AI Gateway）
- 非游戏、非算法设计，聚焦数据管理与分析

## 🗃️ 数据库设计 (SQLite)

### 表结构

```sql
-- 类别表（预置 + 用户自定义）
CREATE TABLE Categories (
    Id INTEGER PRIMARY KEY AUTOINCREMENT,
    Name TEXT NOT NULL UNIQUE,
    Type TEXT NOT NULL CHECK(Type IN ('Income', 'Expense')),  -- 收入/支出
    Icon TEXT
);

-- 收支记录表
CREATE TABLE Transactions (
    Id INTEGER PRIMARY KEY AUTOINCREMENT,
    Amount REAL NOT NULL,           -- 金额（支出为正，收入为正，用Type区分）
    Type TEXT NOT NULL CHECK(Type IN ('Income', 'Expense')),
    CategoryId INTEGER NOT NULL,
    TransactionDate TEXT NOT NULL,  -- ISO 格式: yyyy-MM-dd
    Note TEXT,
    CreatedAt TEXT DEFAULT (datetime('now')),
    FOREIGN KEY (CategoryId) REFERENCES Categories(Id)
);

-- 索引优化查询
CREATE INDEX idx_transactions_date ON Transactions(TransactionDate);
```

## 📈 ScottPlot 集成示例

在 ViewModel 中更新图表的常见方式：通过事件或直接操作控件。为简单起见，可在 code-behind 中调用封装的服务。

**ChartService.cs**：

```csharp
using ScottPlot;

public class ChartService
{
    public void PlotMonthlyExpense(WpfPlot plot, List<Transaction> expenses)
    {
        plot.Plot.Clear();
        // 按月份分组聚合
        var monthly = expenses
            .GroupBy(t => new { t.TransactionDate.Year, t.TransactionDate.Month })
            .Select(g => new { Month = g.Key.Month, Total = g.Sum(t => t.Amount) })
            .OrderBy(x => x.Month)
            .ToList();
        
        double[] xs = monthly.Select(x => (double)x.Month).ToArray();
        double[] ys = monthly.Select(x => x.Total).ToArray();
        
        plot.Plot.AddScatter(xs, ys);
        plot.Plot.Title("月度支出趋势");
        plot.Plot.XLabel("月份");
        plot.Plot.YLabel("金额（元）");
        plot.Refresh();
    }
}
```

## ☁️ 云端 API 集成（DeepSeek）

### 1. 获取 API Key

推荐使用 **Vercel AI Gateway**（注册即送 $5/月，调用 DeepSeek）：

- 访问 [vercel.com/ai-gateway](https://vercel.com/ai-gateway)
- 用 GitHub 登录 → 创建项目 → 获取 API Key（格式：`vc-xxx...`）
- 兼容 OpenAI 接口，Base URL: `https://gateway.ai.vercel.app/v1/chat/completions`

或者直接使用 DeepSeek 官方（需充值少量余额）。

### 2. 配置文件 `appsettings.json`

```json
{
  "AiApi": {
    "BaseUrl": "https://gateway.ai.vercel.app/v1/chat/completions",
    "ApiKey": "你的API密钥",
    "Model": "deepseek/deepseek-chat"   // Vercel 路由名称
  }
}
```

### 3. 读取配置


### 4. AI Service 实现

```csharp
using System.Text;
using System.Text.Json;

public class AiService : IAiService
{
    private readonly HttpClient _httpClient;
    private readonly string _apiKey;
    private readonly string _baseUrl;
    private readonly string _model;
    
    public AiService(IConfiguration config)
    {
        _apiKey = config["AiApi:ApiKey"];
        _baseUrl = config["AiApi:BaseUrl"];
        _model = config["AiApi:Model"];
        _httpClient = new HttpClient();
        _httpClient.DefaultRequestHeaders.Add("Authorization", $"Bearer {_apiKey}");
    }
    
    public async Task<string> GetFinancialAdviceAsync(List<Transaction> recentTransactions)
    {
        // 构建 Prompt
        var prompt = BuildPrompt(recentTransactions);
        
        var requestBody = new
        {
            model = _model,
            messages = new[]
            {
                new { role = "system", content = "你是一位财务分析助手，请根据用户近期收支数据给出简洁、实用的理财建议。" },
                new { role = "user", content = prompt }
            },
            temperature = 0.7,
            max_tokens = 500
        };
        
        var json = JsonSerializer.Serialize(requestBody);
        var content = new StringContent(json, Encoding.UTF8, "application/json");
        
        var response = await _httpClient.PostAsync(_baseUrl, content);
        response.EnsureSuccessStatusCode();
        
        var responseJson = await response.Content.ReadAsStringAsync();
        using var doc = JsonDocument.Parse(responseJson);
        var result = doc.RootElement.GetProperty("choices")[0].GetProperty("message").GetProperty("content").GetString();
        return result;
    }
    
    private string BuildPrompt(List<Transaction> transactions)
    {
        var totalExpense = transactions.Where(t => t.Type == "Expense").Sum(t => t.Amount);
        var topCategory = transactions.Where(t => t.Type == "Expense")
            .GroupBy(t => t.CategoryId)
            .OrderByDescending(g => g.Sum(t => t.Amount))
            .FirstOrDefault()?.Key;
        
        return $"近一周支出总计：{totalExpense}元。主要支出类别ID：{topCategory}。请给出改善建议。";
    }
}
```

## 🚀 运行与调试

- 将 `appsettings.json` 属性设置为“如果较新则复制”
- 确保 `finance.db` 文件自动创建（首次运行数据库服务时会建表）
- API 调用建议在后台线程执行，避免阻塞 UI（使用 `async/await`）

```csharp
// 示例：异步调用 AI 分析
private async void OnAnalyzeClicked(object sender, RoutedEventArgs e)
{

    {

    {
        MessageBox.Show($"调用失败：{ex.Message}");

    finally
    {
        LoadingIndicator.Visibility = Visibility.Collapsed;
    }
}
```
