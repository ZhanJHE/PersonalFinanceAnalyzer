# 技术功能改动日志

> 记录项目开发过程中的所有功能新增、改进和问题修复。

---

## v1.0.0 — 2026-06

### 项目骨架搭建
- 创建 WPF 客户端项目（.NET 9 + MVVM）
- 创建 ASP.NET Core 后端项目（.NET 10）
- 搭建解决方案结构（PersonalFinanceAnalyzer.slnx）
- 配置依赖注入（DI）容器
- 安装 ScottPlot.WPF 5.x 图表库、CommunityToolkit.Mvvm、Microsoft.Data.Sqlite

### 数据层
- 设计 Transaction（GUID 主键）、Category 数据模型
- 实现本地 SQLite 数据库自动建表 + 预置 8 种类别
- 实现完整 CRUD（增删改查）操作
- 实现 CSV 通用导入 + 微信账单专用解析器（WeChatBillParser）
- 新增微信 XLSX 账单解析器（支持微信最新导出格式 .xlsx）
- 新增 SQLite DailyView 视图，按天聚合交易数据用于图表快速查询
- 新增 Budget（预算）模型和数据库表

### 服务层
- DatabaseService：本地 SQLite 数据访问
- TransactionService：离线/在线路由抽象层
- CloudDataService：云端 HTTP API 调用
- SyncService：数据同步服务
- ServerAiService：AI 分析代理（经后端中转）
- ChartService：ScottPlot 图表渲染封装
- CategoryService：类别管理
- ExcelExportService：ClosedXML Excel 导出
- WeChatXlsxBillParser：微信 XLSX 账单解析
- DailyViewService：日汇总数据读取
- BudgetService：预算管理
- AuthService：JWT 登录认证 + DPAPI Token 持久化
- CsvImportService：通用 CSV + 微信账单自动识别路由

### 后端服务
- 创建 ASP.NET Core Web API 项目
- AuthController：JWT 登录/注册（BCrypt 密码哈希）
- AiController：AI 代理转发 + 限额检查（免费 10 次/月，会员 100 次/月）
- SyncController：数据上传/下载接口
- QuotaService：使用次数限额逻辑
- JwtService：JWT 令牌生成
- AppDbContext：EF Core + SQLite 数据上下文
- 用户模型（User）、交易模型（ServerTransaction）、使用日志（AiUsageLog）

### 界面功能
- 主窗口：菜单栏 + 4 个 Tab（概览/记录/分析/预算）+ 底部状态栏
- DashboardView：本月收入/支出/结余卡片 + 近期交易列表
- TransactionView：交易记录 DataGrid + 新增表单 + 筛选搜索 + 类别管理
- AnalysisView：收支净值折线图 + 支出分类占比饼图（1/3/6 月三个独立 Tab）
- BudgetView：预算设置 + 进度条 + 超限提醒
- LoginWindow：登录/注册弹窗（含自定义服务器地址输入框）
- AboutWindow：关于窗口（作者 GitHub 主页、仓库链接、邮箱）
- AiReportWindow：AI 分析报告弹窗（Markdown 渲染）
- EditTransactionWindow：编辑记录弹窗
- SyncConflictWindow：同步冲突解决窗口（保留本地/保留云端/逐条处理三选一）
- WeChatImportGuide：微信导入引导窗口
- ServerConfigWindow：服务器连接配置窗口

### 图表可视化
- 收支净值折线图：单条净值线 + 0 轴分界（绿色盈余、红色亏损）
- 三个时间范围独立 Tab（最近一个月 / 最近三个月 / 最近半年）
- 固定 15 个数据点，自适应时间跨度
- 每个数据点标注金额数值
- 支出分类占比饼图，扇形标注百分比
- 饼图颜色对应类别（粉红=餐饮、蓝色=购物等）
- 禁用鼠标缩放/拖拽交互（静态展示）
- 图表刷新 Toast 横幅通知

### AI 智能分析
- 后端 DeepSeek API 代理转发（API Key 在服务端）
- 三种分析范围：最近一个月 / 最近三个月 / 最近半年
- 三种不同的 AI Prompt 模板（月度总结、季度分析、半年总结）
- 使用次数限额（免费 10 次/月，会员 100 次/月）
- Markdown 格式化渲染（MdXaml 库）
- 未登录时弹窗提示并跳转登录

### 数据同步
- SHA256 哈希比对快速检查同步状态
- 哈希不一致时弹出三选一窗口（保留本地 / 保留云端 / 逐条处理）
- 登录后自动同步
- 退出时询问是否同步
- 同步状态栏显示

### 数据导入导出
- 通用 CSV 导入
- 微信 XLSX 账单导入（含使用引导）
- Excel 导出（ClosedXML，含合计公式）
- 数据库备份 / 恢复

### 安全加固
- BCrypt 密码哈希（替代 SHA256）
- JWT 认证令牌
- HTTPS 通信加密
- Rate Limiting 限流（登录 10 次/分钟、注册 5 次/分钟、AI 30 次/分钟）
- DPAPI Token 本地加密持久化
- 服务端 API Key 不暴露给客户端

### 基础设施
- NSIS 安装包（中文界面、自动杀旧进程、卸载询问删数据）
- Serilog 服务端日志（控制台 + 滚动文件）
- 启动画面（SplashScreen）
- 快捷键（Ctrl+N 新增、Ctrl+F 搜索、Ctrl+E 导出、F5 同步）
- 全局异常处理
- 数据库路径改为 %APPDATA%（安装到 Program Files 后可写）

### 开发支持
- MySQL 开发数据库自动重置（appsettings.Development.json 配置）
- 种子数据（测试用户张三/123456 + 9 条示例交易）
- 单元测试（xUnit）：WeChatBillParser 6 个 + QuotaService 5 个 = 11 个测试

### 问题修复
- CSV 导入金额精度（decimal.TryParse + InvariantCulture）
- 图表中文字体方框问题（改用 ScottPlot.Fonts.Default + 内嵌字体回退）
- 饼图百分比标签位置反转（Y 轴取反）
- XAML 中 Ellipse 兼容性导致闪退（改为 Border）
- 旧数据库缺少 UpdatedAt 列（ALTER TABLE 迁移）
- 旧数据库缺少 Color 列（ALTER TABLE 迁移）
- 种子数据重复导入（固定 GUID）
- QuotaService FindAsync 参数错误
- 分析视图 PropertyChanged 处理器和显式刷新冲突
- 分类新增时 Color 列遗漏
- 启动时不再检查服务器连接（离线可用）
- AI 三种范围返回相同结果（ReportType 硬编码修复）
